How to push a website to GitHub

There are two main types of website:

  • Command Line Interface (CLI)
  • Graphical User Interface

To push a website to GitHub (an online code repository), we use CLI.

Steps to Follow

  1. Go to GitHub and download GitHub desktop
  2. Then, log into your GitHub account on GitHub
  3. Make a new repository by pressing the "+" in the top corner and selecting "New Repository"
  4. Adjust your repository to fit your needs. Make sure to check "initalize this repository with README.md"
  5. Open your command line
  6. At the same time, create a folder with the desired name
  7. Once in command line, use the command "DIR" to display the files within your directory
  8. Go into the desired folder/directory with the "CD" command. Then, create a file with the "MD" command for thh README
  9. Then add it to Git with the git commands: "git add README.md"
  10. Then, do "git -m commit "added README". This creates a log of what was done.
  11. To finalize, do "git remote add origin "insert website repository link"" to connect the repository to the website. Log in if it prompts you
